**HARDLINT** is a powerful, lightweight, and professional-grade OSINT (Open Source Intelligence) tool designed for researchers, penetration testers, and privacy enthusiasts. It allows you to gather information from public sources with a focus on speed, precision, and ease of use.

![Hardlint Logo](https://via.placeholder.com/800x200?text=HARDLINT+OSINT+TOOL)

## üöÄ Features

- **Massive Username Search**: Scans over 100+ social networks, gaming platforms, and developer sites (GitHub, TikTok, Tinder, Spotify, etc.).
- **Email Analysis**:
    - Automatic **Gravatar** profile detection.
    - Specialized **Google Dorks** generation for manual breach search.
- **IP Geolocation**: Detailed location data (Country, City, ISP, Coordinates) with direct Google Maps integration.
- **Enhanced Phone Lookup**: Automatic parsing and formatting with direct links to **TrueCaller**, **Sync.me**, and **Tellows**.
- **Interactive CLI**: Beautiful color-coded interface with a retro hacker aesthetic.

## üìñ Usage

Once started, use the following command structure:

- `search <<username "<name>"`: Search for a username across 100+ sites.
- `search <<email "<email>"`: Analyze an email address.
- `search <<number "<phone>"`: Perform a reverse phone lookup.
- `search <<ip "<ip_address>"`: Geolocate an IP address.
- `!help`: Show the help menu.
- `exit`: Close the tool.
